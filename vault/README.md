При первичном добавлении ликвидности в Port (deposit) пользователю создается нфт, в которой лежит Lp токен для опредления его доли и инфа о росте наград. Перед заливкой ликвы все награды должны быть обновлены через update_position_reward и update_pool_reward.
Ликвидность можно добавить в нфт через increase_liquidity, перед этим нужно обновить и склеймить все награды (clame_position_reward и claim_pool_reward )
Ликвидность можно забрать через withdraw, где также награды должны быть обновлены и склеймлены.
